# Instrucciones

## configuracion

1 - crear una carpeta "config" con un archivo default.json de forma /backend/config/default.json

2 - default.json debe tener la forma:

````json
{
  "port": 3333
}
````

## Inicializacion
> npm i

## Usar (en desarrollo)
Elegir npm o pnpm
> npm run dev

> pnpm run dev
